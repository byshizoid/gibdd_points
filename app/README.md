# Desktop-приложение (Windows)

Это GUI-версия для подсчёта баллов по папкам и генерации рапорта на повышение.

## Что учитывается
Баллы считаются по подпапкам в выбранной папке (по названиям папок):

- штрафы (10)
- аресты (15)
- тонировка (5)
- поставки (25)
- лишение_ву (15)
- аннулирование_ву (15)
- выдача_ву (15)
- выдача_грз (15)
- эвакуация (10)
- гмп (50)
- мероприятие (30)

**ДТП и пост 30 минут не учитываются** (по текущему ТЗ).

## Запуск
1. Установи Python 3.10+.
2. Открой папку `app`.
3. Запусти `run_gui.bat`.

При первом запуске установится PySide6.

## Как пользоваться
- Вкладка **Папки** → выбрать папку с подпапками категорий.
- Вкладка **Сотрудник** → заполнить данные.
- Вкладка **Рапорт** → скопировать или сохранить в файл.

Настройки сохраняются в `app/_data/config.json`.
